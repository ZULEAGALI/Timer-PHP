# Timer-Python
La aplicación cuenta con un temporizador para sesiones de trabajo, descansos cortos y descansos largos, y también realiza un seguimiento del número de "timers" completados. La aplicación se creó utilizando el kit de herramientas GUI de Tkinter y temas de ttkbootstrap.
Al clonar instalar:   
pip install -r requirements.txt
crear y activa el entorno virtual : python3 -m venv .venv , source .venv/bin/activate
